# ChatSphere

Real-time chat application with a NestJS backend and Next.js frontend.

## Project structure

```
ChatSphere/
├── chatsphere-backend/   # NestJS API server
└── chatsphere-frontend/  # Next.js web app
```

## Getting started

### Backend

```bash
cd chatsphere-backend
npm install
npm run start:dev
```

### Frontend

```bash
cd chatsphere-frontend
npm install
npm run dev
```
